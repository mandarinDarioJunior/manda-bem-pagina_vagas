Sobre o Processo Seletivo

O objetivo do nosso processo seletivo é desenvolver estagiários para alcançarem a sua melhor versão.

Teste Remoto

Junto com um formulário simples de inscrição, enviaremos um teste que deverá ser executado até o final do período de inscrições!

Cadastro do Currículo e Análise Comportamental

Os aprovados na etapa 1 farão o cadastro completo de currículo e análise comportamental dentro de uma plataforma parceira, onde daremos sequência ao processo seletivo!

Dinâmicas

Diferentes dinâmicas – em grupo e/ou individual – para cada vaga. Os aprovados nesta etapa irão para nossa fase final!

Teste e entrevista com o gestor

Última etapa! Teste ao vivo e um bate-papo com os líderes da área aplicada, para nos conhecermos melhor.

Perguntas Frequentes

// O estágio será presencial?

No momento, estamos todos em regime Home office, mas consideramos voltar ao escritório quando for seguro para todos.

// Inglês é obrigatório?

Não, para nenhuma vaga.

// Qual a carga horária semanal do estágio?

30h semanais.

// Qual o período do estágio?

Temos dois turnos de estágio, um no período da manhã para estudantes noturnos e outro no período da tarde, para estudantes matutinos.

// Qual a previsão de início do estágio?

Agosto/2021.

// Já sou formadx. Posso aplicar para o programa de estágio?

Não. De acordo com a Legislação do Estágio no Brasil, apenas alunos de instituições educacionais podem fazer estágio.

// Como saberei se fui aprovadx em cada etapa?

Enviaremos um email e/ou mensagem via WhatsApp comunicando a sua aprovação ou não no final de cada fase e explicando os próximos passos.

// O processo terá alguma fase presencial?

É possível, para algumas vagas, que a última etapa seja um teste presencial juntamente com entrevista com o gestor da área. Informaremos com antecedência e, caso ocorra, tomaremos todas as medidas de higiene e distanciamento para garantir a segurança de todos.

// Estou com outras dúvidas. Onde posso entrar em contato?

Envie um email para vagas@mandarin.com.br
